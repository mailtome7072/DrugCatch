# 🗺️ 프로젝트 플랜 확장판

## 1. 프로젝트 개요
- **목표:** 처방전 또는 약품 조제약 정보를 활용한 복약정보 조회 웹 MVP 개발
- **기간:** 2일 (Phase 1 완료 기준)
- **범위:** 이미지 입력 → 분석 → 결과 조회까지의 기본 흐름 구현

---

## 2. 핵심 기능
- 📋 사용자 경고 및 동의 UI (개인정보 및 전문가 확인 안내)
- 📋 이미지 입력 방식 선택 (촬영 vs 업로드)
- 📋 이미지 미리보기 및 분석 시작 버튼
- 📋 OCR 기반 처방전 텍스트 추출
- 📋 약품 이미지 인식 및 약품명 추출
- 📋 데이터 파싱 및 구조화
- 📋 분석 완료 후 결과 화면 (병명 → 증상 → 약품 카드)
- 📋 에러 처리 및 로깅

---

## 3. 사용자 시나리오
1. **사용자 동의**
   - 사용자는 개인정보 처리 및 전문가 확인 안내를 읽고 동의 체크박스를 선택한다.
2. **이미지 입력**
   - 사용자는 처방전 또는 약품 사진을 촬영하거나 기기에서 업로드한다.
3. **미리보기**
   - 업로드된 이미지를 확인하고 필요 시 교체한다.
4. **분석 시작**
   - "이미지 분석 시작" 버튼을 클릭하면 OCR/이미지 인식이 진행된다.
   - 로딩 인디케이터가 표시된다.
5. **결과 확인**
   - 분석 완료 후 인디케이터 종료
   - 결과 화면에 **주요 병명 → 증상 → 약품 정보 카드** 순으로 표시된다.
   - 사용자에게 "최종 확인은 의사·약사에게"라는 안내가 제공된다.

## 4. 기술 스택
- **Frontend**
  - React.js / Next.js
  - TailwindCSS 또는 Material UI
- **Backend**
  - Node.js (Express)
  - Python (OCR 및 이미지 처리 모듈)
- **OCR/이미지 처리**
  - Tesseract OCR
  - OpenCV / TensorFlow Lite
- **CI/CD**
  - GitHub Actions
  - Docker
- **Infra**
  - AWS (EC2, S3, RDS) 또는 Vercel/Netlify
